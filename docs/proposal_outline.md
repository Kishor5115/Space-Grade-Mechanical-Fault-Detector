# Proposal Outline

## Project Title

Space-Grade Mechanical Fault Detector

## Competition

SSCS Chipathon 2026 — Track B

## Team

Team Space Jam

## Objective

Design a radiation-tolerant ASIC capable of autonomous spacecraft vibration fault detection using fixed-point spectral analysis.

## Key Components

* SPI ADC Interface
* Goertzel DSP Core
* Fault Detection Logic
* Debounce Filtering
* Triple Modular Redundancy (TMR)

## Target Technology

GlobalFoundries GF180MCU

## Current Status

Architecture definition and RTL planning.
